# Getting Started

Installation and setup guide for the component library.

## Installation

### 1. Install Dependencies

```bash
npm install \
  react@19.2.0 \
  react-dom@19.2.0 \
  @base-ui/react@1.0.0 \

  class-variance-authority@0.7.1 \
  clsx@2.1.1 \

  tailwindcss@4.1.18 \
  @tailwindcss/vite@4.1.18 \

  react-resizable-panels@4.2.0 \
  recharts@2.15.4 \
  sonner@2.0.7 \
  vaul@1.1.2
```

### 2. Configure Vite

Add Tailwind plugin to `vite.config.ts`:

```typescript
import { defineConfig } from 'vite'
import react from '@vitejs/plugin-react'
import tailwindcss from '@tailwindcss/vite'
import path from 'path'

export default defineConfig({
  plugins: [react(), tailwindcss()],
  resolve: {
    alias: {
      '@': path.resolve(__dirname, './src'),
    },
  },
})
```

### 3. Configure TypeScript

Update `tsconfig.json`:

```json
{
  "compilerOptions": {
    "baseUrl": ".",
    "paths": {
      "@/*": ["./src/*"]
    }
  }
}
```

### 4. Create Utils

Create `src/lib/utils.ts`:

```typescript
import { clsx, type ClassValue } from "clsx";
import { twMerge } from "tailwind-merge";

export function cn(...inputs: ClassValue[]) {
  return twMerge(clsx(inputs));
}
```

## Customization

### Change Colors

Edit CSS variables in `index.css`:

```css
:root {
  --primary: #171717;      /* Primary color */
  --accent: #86EFAC;       /* Accent color */
  --destructive: #DC2626;  /* Error color */
}
```

### Add Border Radius

```css
:root {
  --radius: 0.5rem;  /* Global border radius */
}
```

### Change Font

```css
:root {
  font-family: "Your Font", monospace;
}

@theme inline {
  --font-sans: "Your Font", monospace;
}
```

## Troubleshooting

### Import errors with `@/` alias
- Check `tsconfig.json` paths configuration
- Restart TypeScript server
- Verify Vite config includes alias

### Styles not working
- Verify all CSS imports in `index.css`
- Check CSS variables are defined
- Ensure Tailwind plugin is loaded

### Missing dependencies
- Run `npm install` again
- Clear node_modules and reinstall
